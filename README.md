# Smart Calculator Project

Welcome to the Smart Calculator project! This repository contains a fully functional, GUI-based Smart Calculator built in two different languages: **Java** and **Python**.

## 🎨 Features
Both versions of the calculator feature an identical, modern, and easy-to-use user interface. 
They support:
- **Basic Arithmetic:** Addition (`+`), Subtraction (`-`), Multiplication (`*`), and Division (`/`).
- **Advanced Mathematics:** 
  - Trigonometry: `sin`, `cos`, `tan`
  - Logarithms: `log`
  - Roots and Powers: `sqrt`, exponenents (`^`)
  - Factorials: `!`
  - Constants: `pi`
- **Utility Functions:** Clear all (`C`) and Delete last character (`DEL`).

---

## 🐍 Python Version
The Python version is built using the built-in `tkinter` library for the graphical interface and the `math` module for evaluating complex mathematical expressions.

### Requirements
- Python 3.x

### How to Run
1. Navigate to the `python` directory:
   ```bash
   cd python
   ```
2. Run the script:
   ```bash
   python Calculator.py
   ```

---

## ☕ Java Version
The Java version is built using `Swing` and `AWT` for the graphical user interface. It also includes a custom-built recursive descent parser to evaluate mathematical expressions passed as strings safely and effectively without relying on an external engine.

### Requirements
- Java Development Kit (JDK) 8 or higher.

### How to Run
1. Navigate to the `java` directory:
   ```bash
   cd java
   ```
2. Compile the Java file:
   ```bash
   javac Calculator.java
   ```
3. Run the compiled application:
   ```bash
   java Calculator
   ```
